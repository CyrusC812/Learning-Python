# 2: Strings - Working with Textual Data
## General
- Triple quotation marks allows multi-line strings
- Variables can be used to store strings
- Count can be used to count the amount of characters found in string and output it
- find can be used to output the first index of the wanted word
- replace can be used to replace a snippet of a string to another
## String manipulation
- .upper() turns the whole text upper cased
- .lower() turns the whole text lower cased
## concatenation
- fstrings are more readable than standard + method
- fstrings allow .upper or similar added to its place holders

## Additional
- Use help for extra info , e.g. print(help(str))
- Use dir to list all possible methods e.g. print(dir(str))
